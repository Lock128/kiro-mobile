# Product: kiro_flutter_auth

Cross-platform Flutter mobile app providing authentication and session management for the Kiro platform. Targets Android, iOS, and Web.

## Core Capabilities
- WebView-based OAuth sign-in flow (mobile) and popup-based sign-in (web)
- Secure credential storage (Keychain on iOS, Keystore on Android, encrypted localStorage on web)
- Session and task management views
- Network connectivity monitoring with offline overlay
- Three-tab navigation: Create, Chats, Tasks

## Status
Alpha — not security-audited. Not published to pub.dev.
